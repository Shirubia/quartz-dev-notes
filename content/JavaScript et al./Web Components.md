>**Web Components** is a set of technologies for creating reusable and encapsulated **custom** HTML **elements** in modern web development.

## Tecnologías

* Custom elements
* Shadow dom
* Templates y slots

## Angular elements

>_Angular elements_ are Angular components packaged as _custom elements_ (also called Web Components), a web standard for defining new HTML elements in a framework-agnostic way.
